# 자동차 경주

# 기능 목록

- 게임 세팅
  - [x] 자동차 이름 입력
  - [x] 시도할 횟수 입력

- 게임 로직 처리
  - [ ] 자동차 별로 무작위로 전진
  - [ ] 이동 후 차수별 실행 결과(자동차의 위치) 출력
  - [ ] 차수 횟수 도달시 게임 종료

- 게임 종료시
  - [ ] 가장 멀리 간 자동차들이 우승
  - [ ] 우승자 안내 문구 출력
